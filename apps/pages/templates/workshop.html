Open Data Cube Workshop

Welcome to the ODC Workshop at IGARSS. This page serves to aggregate some useful links and content that may be referenced during this presentation. You can use this page to follow along or to reference at a later date. You will receive a sheet containing a server IP address that will serve as a sandboxed Data Cube instance for this workshop.

When directed to visit a webpage or open a configuration file or document, you can use this page to quickly and easily open the files. Additionally, a command 'cheat sheet' is provided to ensure that everyone who wants to follow along is able to keep up. For ease of use, this has been divided into sections.

Command Cheat Sheet

SSH into the server:
ssh localuser@{Your server IP address}
Activating a virtual environment:
source ~/Datacube/datacube_env/bin/activate
Adding a dataset type/product definition:
datacube -v product add ~/Datacube/agdc-v2/ingest/dataset_types/gpm/gpm_imerg_gis.yaml
Running a preprocessing script on a dataset:
python ~/Datacube/agdc-v2/ingest/prepare_scripts/gpm/gpm_imerg_gis_prepare.py /datacube/original_data/GPM/Month/3B-MO*
Adding dataset to the Data Cube:
datacube -v dataset add /datacube/original_data/GPM/Month/3B-MO*/*.yaml
Running the ingestion process:
datacube -v ingest -c ~/Datacube/agdc-v2/ingest/ingestion_configs/gpm/gpm_monthly_global.yaml
